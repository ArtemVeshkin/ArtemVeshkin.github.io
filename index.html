Frequently Asked Questions

What platforms is offered for users?

We provide platforms based on open source software and mostly open source hardware, namely Linux distributions and RISC-V processors. There are several builds that serves different purposes: The W, O, G and P types, which are described below.

What is RISC-V?

RISC-V is modern, designed from scratch instruction set architecture, which is simpler, faster and more flexible ISA than existing alternatives. It also features higher code density than wide-spread x86 architecture despite being RISC.

Does it really works?

It is new but production ready and supported by Linux kernel and most software. It's also good for computing.
